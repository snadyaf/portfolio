# Portfólio Snadyaf

Este portfólio apresenta três projetos reais em JavaScript, incluindo um sistema fullstack, um serviço backend e uma ferramenta de automação.

## Como rodar localmente

1. Instale dependências de teste:

```bash
npm install
```

2. Abra o portfólio em um servidor local:

```bash
npx http-server . -p 8080
```

3. Execute os testes Cypress:

```bash
npm test
```

4. Ou abra a interface do Cypress:

```bash
npm run cypress:open
```

## Arquivos importantes

- `index.html` — página principal do portfólio
- `style.css` — estilo customizado com tema Light/Dark
- `script.js` — alternância de tema e animação de seções
- `relatorio.md` — explicação detalhada dos projetos
- `projects/` — projetos em JavaScript com código de exemplo
- `cypress/` — testes automatizados
