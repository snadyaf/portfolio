# Helpdesk inteligente

Uma aplicação fullstack em JavaScript com frontend leve e backend Node.js/Express.

## Descrição
O projeto implementa um sistema de tickets com criação de chamados, listagem de atendimentos e armazenamento em memória para demonstração.

## Como rodar

```bash
cd projects/fullstack-helpdesk
npm install
node server.js
```

Abra `http://localhost:3000` no navegador.
